# KI-Assistent für Zahnarztpraxen — Landingpage

Landingpage (Freebie-Funnel) für die Zielgruppe **Zahnärzte**.
Marke: **The Power of AI**. Statische Single-File-Seite (`index.html`).

## Aufbau
Hero → Praxisalltag → Lösung (KI-Assistent) + 3 Schritte → Freebie-Opt-in (Name/Praxis/E-Mail/Telefon) → Call-Buchung nach Absenden → Datenschutz → FAQ → Footer.

## Noch scharfzuschalten
- `CALENDLY_URL` im `<script>` → echter Buchungslink.
- Formular an **Brevo** anbinden (Lead-Capture, aktuell Demo-Verhalten).

## Deploy
Statisch — `index.html` liegt im Root. Direkt via Netlify/GitHub Pages veröffentlichbar.
